# desafio-combate-oo-java-devsuperior
Solução do desafio “Combate OO” (DevSuperior). Instancia dois campeões com vida, ataque e armadura. Executa turnos de combate aplicando dano mínimo de 1, exibe o status a cada rodada e encerra quando algum campeão morre. Implementação OO com classe Champion, regras de dano e atenção ao caso-limite (vida não negativa).
